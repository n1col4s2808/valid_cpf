# Validador de CPF 

Aplicação web desenvolvida com Python e Flask que valida CPFs em tempo real.

## Funcionalidades
- Valida se o CPF tem 11 dígitos
- Aceita apenas números
- Bloqueia sequências repetidas (ex: 111.111.111-11)
- Calcula os dígitos verificadores pela fórmula oficial da Receita Federal
- Interface web

## Como usar
Execute o arquivo no terminal:
python app.py

Acesse no navegador: http://127.0.0.1:5000

## Tecnologias
- Python 3
- Flask
- HTML e CSS
